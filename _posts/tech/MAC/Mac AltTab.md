>   **AltTab** 是一款 macOS 应用，它可以让你的 Mac 像 Windows 一样使用 Alt + Tab 切换窗口时，可以预览窗口内容，快捷键是 option ⌥ + tab。

![](https://curtisyan.oss-cn-shenzhen.aliyuncs.com/img/no_important/202408060948857.jpg)

<br>

## AltTab：Windows alt-tab on macOS

就像上图的一样，当按下快捷键（option⌥ + tab）之后，你就能看到和 Windows 类似的窗口切换界面，操作逻辑也完全相同，按住 option 的同时继续按 tab 可以在不同窗口间切换。

比如官网的这个图：

![](https://curtisyan.oss-cn-shenzhen.aliyuncs.com/img/no_important/202408060951188.jpg)

AltTab 拥有中文界面，并且有着非常…详细的设置页面，用来控制、自定义界面，简直比 Windows 还要详细：

![](https://curtisyan.oss-cn-shenzhen.aliyuncs.com/img/no_important/202408060952053.jpg)

![](https://curtisyan.oss-cn-shenzhen.aliyuncs.com/img/no_important/202408060952213.jpg)

AltTab 需要两个系统权限：

- 辅助功能（使用快捷键切换窗口）
- 屏幕录制（显示窗口截屏与标题）
